# FireCamp v3.2.3 Released

We are happy to announce the release of FireCamp v3.2.3! This release includes several new features, improvements, and bug fixes contributed by our amazing open source community.

## What's New

- Added Twitter and GitHub star badges to showcase FireCamp's growth ([#52](https://github.com/firecamp-dev/firecamp/pull/52), [#54](https://github.com/firecamp-dev/firecamp/pull/54))
- Improved UI with a new color scheme selector ([#72](https://github.com/firecamp-dev/firecamp/pull/72)) 
- Switched to Mantine as the UI library for a refreshed look and feel ([#83](https://github.com/firecamp-dev/firecamp/pull/83))
- Added a FileInput component for better file uploads ([#49](https://github.com/firecamp-dev/firecamp/pull/49))
- Launched `@firecamp/types` package for shared type definitions ([#90](https://github.com/firecamp-dev/firecamp/pull/90))
- Added tooltip component and tooltip on action bar items ([#104](https://github.com/firecamp-dev/firecamp/pull/104))
- Upgraded to React 18 ([#123](https://github.com/firecamp-dev/firecamp/pull/123))

## Bug Fixes 

- Fixed Windows build issue due to incorrect escaping in scripts ([#71](https://github.com/firecamp-dev/firecamp/pull/71))
- Fixed Zustand console warnings ([#88](https://github.com/firecamp-dev/firecamp/pull/88))
- Fixed spelling errors ([#92](https://github.com/firecamp-dev/firecamp/pull/92), [#94](https://github.com/firecamp-dev/firecamp/pull/94))

## Other Improvements

- Enhanced documentation ([#50](https://github.com/firecamp-dev/firecamp/pull/50), [#59](https://github.com/firecamp-dev/firecamp/pull/59), [#116](https://github.com/firecamp-dev/firecamp/pull/116), [#125](https://github.com/firecamp-dev/firecamp/pull/125))
- UI optimizations and improvements ([#56](https://github.com/firecamp-dev/firecamp/pull/56), [#58](https://github.com/firecamp-dev/firecamp/pull/58), [#64](https://github.com/firecamp-dev/firecamp/pull/64))  
- Added bulk editing support for form-urlencoded bodies ([#111](https://github.com/firecamp-dev/firecamp/pull/111))

Thank you to everyone who contributed to this release! We appreciate all the PRs from our amazing community. Please keep the contributions coming! 

Get the latest release from GitHub. Let us know if you have any feedback or issues. We look forward to bringing you more awesome improvements soon!
